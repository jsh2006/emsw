이 프로젝트는 Arduino, 서버 프로그램(Processing), 스마트폰 앱(App Inventor)을 모두 연동하여 모니터링과 제어 기능을 동시에 제공하는 완성형 임베디드 시스템을 구현하는 것을 목표로 한다. 시스템 구조는 Arduino가 센서 데이터를 측정하고 액추에이터를 제어하며, 서버는 측정된 데이터를 전달하고 스마트폰 앱과 통신하는 역할을 수행한다. 스마트폰 앱은 사용자 인터페이스를 담당하여 실시간 모니터링 화면과 제어 기능을 제공한다. 본 프로젝트에서 구현하는 주요 기능은 환경 정보를 실시간으로 확인하고, 필요 시 원격으로 장치를 제어하는 통합 IoT 시스템이다.

본 프로젝트의 주제는 “스마트 환경 모니터링 및 자동 환풍기 제어 시스템”이다. Arduino에는 미세먼지 센서(PM2.5), 온습도 센서(DHT11), 가스 센서(MQ-2)가 연결되어 실내 환경 정보를 측정한다. 측정된 값은 Processing 서버를 통해 스마트폰 앱으로 전송되며, 앱에서는 온도, 습도, 미세먼지, 가스 농도를 실시간으로 확인할 수 있다. 또한 사용자는 스마트폰에서 환풍기를 직접 켜거나 끌 수 있으며, 앱의 제어 버튼을 통해 Arduino에 명령이 전달되어 실제 환풍기(DC모터 또는 릴레이)가 작동하도록 한다. 더불어 특정 임계값을 초과할 경우, 예를 들어 가스 농도 상승이나 미세먼지 농도 급증이 감지되면 시스템이 자동으로 환풍기를 동작시키도록 설계한다. 위험 상황에서는 LED와 부저를 통해 경고 신호도 출력된다.# emsw
