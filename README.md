본 프로젝트의 목표는 Arduino 기반 하드웨어, Processing 기반 서버 프로그램, App Inventor로 제작한 스마트폰 애플리케이션을 하나의 통합 시스템으로 구성하여 환경 정보를 실시간으로 모니터링하고, 사용자 명령에 따라 환풍기를 제어할 수 있는 스마트 환경 관리 시스템을 구현하는 것이다. 단순한 센서 측정이 아니라, “모니터링 + 제어” 기능을 모두 포함한 완성형 IoT 구조를 만드는 것을 최종 목적에 둔다.

시스템은 크게 세 부분으로 구성된다. 먼저 Arduino 하드웨어 부분에서는 실내 환경 정보를 측정하기 위해 여러 종류의 센서를 사용한다. 온도·습도는 DHT11 센서를 통해 측정하고, 공기 중 가연성 가스 농도는 MQ-2 센서로 감지하며, 실내 미세먼지 농도(PM2.5)는 전용 먼지 센서를 이용해 수집한다. 각 센서에서 얻은 측정값은 Arduino 내부에서 일정 주기로 갱신되며, 위험 수준에 도달했는지 확인하기 위해 기준값과 비교된다. 또한 환풍기 역할을 하는 DC 모터(또는 릴레이를 통한 환풍기 전원 제어)가 연결되어 있어 자동·수동 제어가 가능하다. 이상 상황이 발생하면 LED와 부저를 활성화하여 경고 신호를 제공하도록 구성한다.

다음으로 Processing 서버 프로그램은 Arduino와 스마트폰 앱 사이의 중간 다리 역할을 수행한다. Arduino는 시리얼 통신을 통해 실시간 환경 데이터를 Processing으로 전송하고, Processing은 이 데이터를 파싱하여 스마트폰 앱에서 접근할 수 있도록 서버 형태로 제공한다. 또한 스마트폰 앱에서 보낸 제어 명령(예: “환풍기 ON”, “환풍기 OFF”)을 수신하면 다시 Arduino로 역전송하여 실제 하드웨어 동작이 이루어지도록 한다. 이 과정에서 서버는 환경 데이터 로그를 PC 화면에 시각적으로 표시하거나 파일 형태로 저장할 수도 있다.

마지막으로 **스마트폰 애플리케이션(App Inventor)**은 최종적인 사용자 인터페이스 역할을 한다. 앱은 Processing 서버에 주기적으로 접속하여 실내 온도, 습도, 미세먼지 농도, 가스 농도 등의 데이터를 실시간 대시보드 형태로 표시한다. 사용자는 화면에 표시되는 버튼을 통해 환풍기를 직접 켜거나 끌 수 있으며, 버튼 조작에 따라 제어 명령이 서버로 전달되고 다시 Arduino로 전송되어 실제 환풍기 모터가 반응한다. 또한 특정 임계치를 초과하는 경우, 예를 들어 미세먼지 농도가 급격히 상승하거나 가스 센서가 위험 수준을 감지하면 앱 화면에 경고 문구를 띄우고, 필요할 경우 알림 기능을 활용하여 사용자에게 주의를 준다.
